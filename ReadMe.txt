Description: Simple asynchronous encryption and decryption program. SetUp.py generates private and public keys,
    Encryption.py will encrypt a text file named text.txt and output text.txt.
    Decryption.py will decrypt ctext.txt and output text.txt.
    Async.py will run SetUp.py, Encryption.py and Decryption.py in order.
How to run: All files should be able to be run with a python interpreter with standard libraries.
    Files need to be in the same directory with ptext.txt.
Files Included: Async.py, Encryption.py, Decryption.py, SetUp.py
